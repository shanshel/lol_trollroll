# 🏆 LoL Synergy Randomizer 2026

A premium, cinematic League of Legends champion selection tool designed for the modern era. Forge your team synergy with a high-end Hextech aesthetic.

![Version](https://img.shields.io/badge/version-2026.1.0-gold)
![License](https://img.shields.io/badge/license-MIT-blue)
![Tech](https://img.shields.io/badge/tech-Vite%20%7C%20Vanilla%20JS-0ac8b9)

## ✨ Key Features

*   **Dynamic Team Building**: Add or remove slots (1-5) on the fly.
*   **Advanced Locking**: Specialize your search by locking champions you want to keep.
*   **Official Assets**: Integrated with Riot's Data Dragon API for official champion artwork.
*   **Cinematic Shuffling**: High-speed "Breathing" roll animation with subtle glow effects.
*   **Complete Roster**: Includes all champions up to the 2026 releases (*Mel Medarda, Yunara, Zaahen*).
*   **Responsive & Fast**: Single-screen experience optimized for performance and zero-lag randomization.

## 🚀 Installation & Local Development

1. **Clone the repo:**
   ```bash
   git clone https://github.com/your-username/combowombo.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the dev server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
